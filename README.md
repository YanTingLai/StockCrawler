## 台灣證券交易所爬蟲
http://www.twse.com.tw

## 使用方式
* 透過R軟體將資料爬下來後存到SQLite中

* Config設定
  * st_day = 1, 起始日期，1為前一日
  * diff_day = 2, 爬取資料日數

因使用條款禁止爬蟲程式，
所以現在很容易被BAN IP。
